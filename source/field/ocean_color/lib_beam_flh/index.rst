Fluorescence line height with Envisat MERIS L1b
===============================================

This tutorial is based on the ESA BEAM Toolbox FLH Operator tutorial where a BEAM processor is implemented in Java.
The application will calculate the fluorescence line height of Envisat MERIS L1b data.
